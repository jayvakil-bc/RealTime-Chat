Real-Time Chat Application (P-T-P)

Total Clients: 2

Technical Implementation:

  • Front-end implemented using HTML, CSS, and JavaScript.
  • Utilizes the Font Awesome library for icons.
  • Integrates the Moment.js library for handling and displaying timestamps.
  • Socket.io employed for real-time communication between clients.
  • The server is built using Node.js and Express.

How it Works:

  • Users connect to the server using Socket.io, and their presence is tracked.
  • Messages and feedback are broadcasted to all connected clients in real-time.
  • The client dynamically updates the UI to display incoming and outgoing messages.
  • Feedback about typing is broadcasted to notify others in the chat.

Usage:

  • Users can access the application through a web browser.
  • Upon entering the site, users can set a display name and start engaging in real-time conversations.
